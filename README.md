# CS50P: Introduction to Programming with Python

This repository contains my solutions to the problem sets from Harvard University's **CS50P** course. The goal is to master Python fundamentals through practical assignments.

## Course Modules Covered
* Functions & Variables: Basic input/output and function definitions.
* Conditionals: Logic gates and decision-making.
* Loops: `while` and `for` iterations.
* Exceptions: Handling errors and user input validation.
* Libraries: Using built-in and external Python packages.
* Unit Tests: Writing tests to ensure code reliability.
* File I/O: Reading from and writing to files.
* Regular Expressions: Advanced pattern matching.
* Object-Oriented Programming: Classes, methods, and properties.

## Environment
* Language: Python 3
* Editor: PyCharm
* Testing: `pytest`
